0: Ex1 selected
1: Ex 1 difficulty level ‘Beginner’ selected
2: ‘Intermediate’ selected
3: ‘Advanced’ selected
4: Ex1 ready to start
5:
6: Ex1 started
7: Ex1 finished
8: Ex1 not successful
9: one repetition successfully done
